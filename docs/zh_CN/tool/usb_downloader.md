## 描述

**ESP32-Downloader** 是一款内置 CP2104 芯片的串口转 USB 烧录工具。

## 参数

*有两个引脚作为预留 ( RTS, DTR )*

|       管脚号       |          管脚名字          |
| :-------------------: |:------------------------: |
|           1           |           GND             |
|           2           |           GPIO0           |
|           3           |           EN              |
|           4           |           TXD             |
|           5           |           RXD             |
|           6           |           3.3V            |
