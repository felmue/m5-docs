## Description

**ESP32-Downloader**  is a USB-to-UART converter. It can be used to download firmware form PC to Microprocessor devices.


TXD led, RXD led, Power led and 6pin @ 2.54mm bus sockets.


## product Feature

- USB typeC-to-Uart


## Include

- 1x M5Stack USB Downloader
